# Instalação do ESM Forum

## Requisitos

- Node.js
- npm
- Git

## Backend

Clone o repositório:

```bash
git clone LINK_DO_FORK_BACKEND
```

Entre na pasta:

```bash
cd esmforum
```

Instale as dependências:

```bash
npm install
```

Execute o backend:

```bash
npm start
```

O servidor será iniciado em:

```text
http://localhost:5000
```

---

## Frontend

Clone o repositório:

```bash
git clone LINK_DO_FORK_FRONTEND
```

Entre na pasta:

```bash
cd esmforum-react
```

Instale as dependências:

```bash
npm install
```

Execute o frontend:

```bash
npm start
```

O sistema será iniciado em:

```text
http://localhost:3000
```
